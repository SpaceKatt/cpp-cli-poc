# cpp-cli-poc

## Development

### Build Toolchain

This project uses `cmake` and `g++` to provide consistent builds.

```bash
apt install cmake g++
```

### Build

A script, `ci-build.sh`, is provided to automate the build process.

```bash
./auto-build.sh
```
